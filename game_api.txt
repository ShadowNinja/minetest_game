minetest_game API
======================
GitHub Repo: https://github.com/minetest/minetest_game

The minetest_game game offers multiple new possibilities in addition to
Minetest's built-in API, allowing you to add new plants, buckets, stairs,
and panes; among other things.  For information on the Minetest API, read [the
Minetest Lua API documentation](https://github.com/minetest/minetest/blob/master/doc/lua_api.txt).

The folliwing mods provide an API, you can find documentation on most of them
inside their mod folder:
  * `default`
  * `beds`
  * `buckets`
  * `creative`
  * `dye`
  * `farming`
  * `stairs`
  * `wool`
  * `xpanes`
